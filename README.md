# Grafiklabbar — DH2323 Computer Graphics Labs

Laboratory assignments for the DH2323 Computer Graphics course at KTH.

## Labs

- **Lab 1** — Image processing (OpenCV): bilinear interpolation, distance transforms, motion blur
- **Lab 2** — OpenGL basics: dot products, lighting, shadow maps
- **Lab 3** — OpenGL advanced: camera positioning, texturing, fill techniques

Each lab directory contains the assignment PDF, reports, solution code, and original skeleton code.
